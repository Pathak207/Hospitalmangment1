Getting Started

After successful installation, you'll need to create your first admin user and configure basic settings.

1. Create Admin User

Use the /api/auth/create-admin endpoint to create your first admin user.

2. Login & Setup

Login with your admin credentials and configure system settings.

  4. 4 Save patient profile

Scheduling Appointments

  1. 1 Go to Appointments section
  2. 2 Click "Schedule Appointment"

  5. 5 Add appointment details and confirm
