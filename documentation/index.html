System Requirements

Minimum Requirements

  • Node.js 18.0 or higher
  • MongoDB 4.4 or higher
  • npm, yarn, or pnpm
  • 2GB RAM minimum

Recommended

  • Node.js 20.0+ for best performance
  • MongoDB Atlas cloud database
  • 4GB RAM or more
  • SSD storage for better performance
